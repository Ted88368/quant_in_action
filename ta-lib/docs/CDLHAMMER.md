# CDLHAMMER（锤头）模式总结

## 模式描述
- 锤头是一个看涨的反转烛台模式，通常出现在下跌趋势中。
- 蜡烛的特点：小实体，下影线长（至少为实体的2倍），上影线短或无。
- 表明在低点有买方力量，价格可能反弹。

## 使用时机
- 当市场处于下跌趋势时，看到锤头可能预示反弹。
- 适合在支撑位附近使用。
- 建议在日线图上使用。

## 配合指标
- **趋势确认**：与移动平均线配合，确认下跌趋势。
- **动量指标**：RSI在30以下时增强信号。
- **波动率指标**：ATR评估波动性。
- **成交量**：成交量放大时增强可靠性。
- **其他烛台模式**：可与Morning Star（启明星）结合。

## 注意事项
- 需要确认锤头的形态。
- 建议结合整体市场环境。
- 并非100%准确，应与其他分析工具结合。
