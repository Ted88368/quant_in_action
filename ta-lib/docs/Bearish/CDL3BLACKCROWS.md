# CDL3BLACKCROWS（三只黑乌鸦）模式总结

## 模式描述
- 三只黑乌鸦是一个看跌的反转烛台模式，通常出现在上涨趋势的顶部。
- 由三根蜡烛组成：
  1. 第一根：长阳线或上涨蜡烛。
  2. 第二根：阴线，开盘价高于第一根的收盘价，收盘价低于第一根的收盘价。
  3. 第三根：阴线，开盘价高于第二根的收盘价，收盘价低于第二根的收盘价。
- 整体显示出市场从乐观到悲观的转变。

## 使用时机
- 当市场处于上涨趋势时，看到三只黑乌鸦模式，可能预示着价格反转下跌。
- 适合在中期趋势中识别可能的顶部，尤其在高位区域。
- 建议在超买区使用，以增加可靠性。

## 配合指标
- **趋势确认**：与移动平均线（如SMA、EMA）配合，确认整体上涨趋势。
- **动量指标**：RSI（相对强弱指数）在70以上时增强看跌信号；MACD在高位死叉时配合。
- **波动率指标**：ATR（平均真实波幅）评估波动性，避免在低波动期误判。
- **成交量**：OBV（平衡成交量）下降时增强可靠性。
- **其他烛台模式**：可与Evening Star（黄昏之星）等其他看跌模式结合使用。

## 判断见顶回落的条件
基于三只黑乌鸦模式及其他指标，判断上涨趋势是否见顶的综合条件可总结为以下公式：

- **基本条件**：三只黑乌鸦形态出现（CDL3BLACKCROWS == 100）
- **增强条件**：
  - 前期处于严重的超买区域（RSI > 75）
  - 三根阴线的实体逐渐增大 [确认杀跌动能加强]
  - 价格下穿重要均线支撑（如 20日均线）
- **综合公式**：
  ```
  见顶信号 = 三只黑乌鸦 AND (
             (成交量放大 且 跌破前 10日支撑位) OR 
             (RSI 出现顶背离 且 价格处于阻力区)
           )
  ```

## 注意事项
- 这是一个反转信号，但需要结合整体市场环境和风险管理。
- 建议在日线或周线图上使用，避免在低时间框架上的噪音。
- 并非100%准确，应与其他分析工具结合。
