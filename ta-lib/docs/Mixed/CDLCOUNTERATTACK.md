# CDLCOUNTERATTACK（反击）模式总结

## 模式描述
- 反击是一个反转烛台模式，可以是看涨或看跌，取决于第二根蜡烛的方向。
- 由两根蜡烛组成：
  1. 第一根：长蜡烛。
  2. 第二根：长蜡烛，与第一根方向相反，开盘价等于第一根的收盘价。
- 显示出市场突然的反击。

## 使用时机
- 当市场处于趋势中时，看到反击模式，可能预示着价格反转。
- 适合在中期趋势中识别可能的转折点。
- 建议在支撑或阻力区域使用。

## 配合指标
- **趋势确认**：与移动平均线（如SMA、EMA）配合，确认整体趋势。
- **动量指标**：RSI（相对强弱指数）在超买/超卖区时增强信号；MACD交叉时配合。
- **波动率指标**：ATR（平均真实波幅）评估波动性。
- **成交量**：OBV（平衡成交量）变化时增强可靠性。
- **其他烛台模式**：可与其他反转模式结合使用。

## 量化识别信号
基于反击模式及其他指标，其看涨（企稳）与看跌（见顶）的量化触发条件如下：

### 1. 看涨信号 (企稳)
- **基本条件**：看涨反击形态出现（CDLCOUNTERATTACK == 100）
- **综合公式**：
  ```
  企稳信号 = 看涨反击 AND (
             (RSI < 30 且 随后日继续收阳确认) OR 
             (成交量在反击当日显著放大 且 触及历史低位)
           )
  ```

### 2. 看跌信号 (见顶)
- **基本条件**：看跌反击形态出现（CDLCOUNTERATTACK == -100）
- **综合公式**：
  ```
  见顶信号 = 看跌反击 AND (
             (RSI > 70 且 价格处于显著阻力位) OR 
             (MACD 出现死叉 且 随后日跳空下行)
           )
  ```

## 注意事项
- 这是一个反转信号，但需要确认第二根蜡烛的方向。
- 建议在日线或周线图上使用。
- 并非100%准确，应与其他分析工具结合。
