# CDLBELTHOLD（捉腰带线）模式总结

## 模式描述
- 捉腰带线是一个反转烛台模式，可以是看涨或看跌，取决于蜡烛的颜色和方向。
- 由一根蜡烛组成：
  - 看涨：长阳线，无上影线，开盘价为最低价。
  - 看跌：长阴线，无下影线，收盘价为最低价。
- 显示出市场单边运动，预示转折。

## 使用时机
- 当市场处于趋势中时，看到捉腰带线模式，可能预示着价格反转。
- 适合在中期趋势中识别可能的转折点。
- 建议在支撑或阻力区域使用。

## 配合指标
- **趋势确认**：与移动平均线（如SMA、EMA）配合，确认整体趋势。
- **动量指标**：RSI（相对强弱指数）在超买/超卖区时增强信号；MACD交叉时配合。
- **波动率指标**：ATR（平均真实波幅）评估波动性。
- **成交量**：OBV（平衡成交量）变化时增强可靠性。
- **其他烛台模式**：可与其他反转模式结合使用。

## 量化识别信号
基于捉腰带线模式及其他指标，其看涨（企稳）与看跌（见顶）的量化触发条件如下：

### 1. 看涨信号 (企稳)
- **基本条件**：看涨捉腰带线形态出现（CDLBELTHOLD == 100）
- **综合公式**：
  ```
  企稳信号 = 看涨捉腰带线 AND (
             (RSI < 25 [极端超卖状态]) OR 
             (成交量显著高于前五日平均值 且 站上 10日均线)
           )
  ```

### 2. 看跌信号 (见顶)
- **基本条件**：看跌捉腰带线形态出现（CDLBELTHOLD == -100）
- **综合公式**：
  ```
  见顶信号 = 看跌捉腰带线 AND (
             (RSI > 75 且 价格处于压力区域) OR 
             (随后一个交易日收阴 且 跌破该长阴线实体中点)
           )
  ```

## 注意事项
- 这是一个强信号，但需要确认蜡烛的方向。
- 建议在日线或周线图上使用。
- 并非100%准确，应与其他分析工具结合。
