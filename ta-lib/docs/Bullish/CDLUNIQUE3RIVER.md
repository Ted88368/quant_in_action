# CDLUNIQUE3RIVER（独特三河）模式总结

## 模式描述
- 独特三河是一个罕见的看涨反转模式，出现在下跌趋势中。
- 由三根蜡烛组成：
  1. 第一根：长阴线。
  2. 第二根：锤头状阴线，下影线创出新低，但收盘价高于第一根阴线的收盘价。
  3. 第三根：短小阳线，其收盘价不高于第二根蜡烛的最高点。
- 它显示了虽然价格创出新低，但卖盘已经无法维持价格在低位，底部正在固化。

## 使用时机
- 出现在连续下跌后的筑底阶段。
- 适合在寻找波段底部的低风险进入点时参考。

## 配合指标
- **成交量**：第二根锤头状阴线伴随放量，而第三根阳线缩量。
- **均线系统**：价格在重要长期均线（如 60日线）获得支撑。
- **动量指标**：RSI 指标底背离。

## 判断下跌企稳的条件
基于独特三河模式及其他指标，判断下跌是否企稳的综合条件可总结为以下公式：

- **基本条件**：独特三河形态出现（CDLUNIQUE3RIVER == 100）
- **增强条件**：
  - 第二根蜡烛的影线极长，显示多头在低位有强力承接
  - 第三根小阳线显示出市场波动性减小，多空趋于平衡
  - 价格处于历史成交密集区或重要支撑位
- **综合公式**：
  ```
  企稳信号 = (独特三河 AND 支撑位确认) OR 
             (独特三河 AND 随后一根长阳线放量突破 10日均线)
  ```

## 注意事项
- 这是一个低频率模式，必须严格满足其三根 K 线的形态要求。
- 确认为重，建议等待价格突破第一根阴线中点或高点后再行介入。
