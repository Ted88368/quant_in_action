# CDLTAKURI（探水竿）模式总结

## 模式描述
- 探水竿（Takuri Line）是一种特殊的看涨反转模式，类似于蜻蜓十字星或长下影的小锤头。
- 特点是下影线极长，实体极小（通常位于价格区间的顶端），几乎没有上影线。
- 它强烈暗示在交易时段内，空头曾试图压低价格，但遭到了多头的猛烈反击，价格最终回升至开盘价附近。

## 使用时机
- 通常出现在急跌或长期下跌趋势的末端。
- 作为一种极端的反转信号，适合在价格回踩深坑时寻找买点。

## 配合指标
- **动量指标**：RSI 在 25 以下的极端超卖区。
- **布林线**：价格跌破布林线下轨后收回。
- **成交量**：成交量在探水竿出现时通常会显著放大，显示“恐慌盘”溢出。

## 判断下跌企稳的条件
基于探水竿模式及其他指标，判断下跌是否企稳的综合条件可总结为以下公式：

- **基本条件**：探水竿形态出现（CDLTAKURI == 100）
- **增强条件**：
  - 下影线长度至少是实体的 3 倍以上
  - 价格处于布林线下轨之外或触及长期均线（如 250日线）支撑
  - 随后一根 K 线为阳线且收盘价高于探水竿的最高价
- **综合公式**：
  ```
  企稳信号 = (探水竿 AND RSI < 25) OR 
             (探水竿 AND 价格后续站上 5日均线 AND 成交量确认)
  ```

## 注意事项
- 下影线的长度决定了信号的强度，越长越好。
- 必须等待下一根 K 线确认，以防止价格惯性下跌。
- 建议结合市场整体情绪 analysis。
