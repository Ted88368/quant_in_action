# CDLSHOOTINGSTAR（流星）模式总结

## 模式描述
- 流星是一个看跌的反转烛台模式，通常出现在上涨趋势中。
- 蜡烛的特点：小实体，上影线长（至少为实体的2倍），下影线短或无。
- 表明在高点有卖方力量，价格可能下跌。

## 使用时机
- 当市场处于上涨趋势时，看到流星可能预示下跌。
- 适合在阻力位附近使用。
- 建议在日线图上使用。

## 配合指标
- **趋势确认**：与移动平均线配合，确认上涨趋势。
- **动量指标**：RSI在70以上时增强信号。
- **波动率指标**：ATR评估波动性。
- **成交量**：成交量放大时增强可靠性。
- **其他烛台模式**：可与Evening Star（黄昏之星）结合。

## 判断见顶回落的条件
基于流星模式及其他指标，判断上涨趋势是否见顶的综合条件可总结为以下公式：

- **基本条件**：流星形态出现（CDLSHOOTINGSTAR == 100）
- **增强条件**：
  - 上影线长度至少是实体的 3 倍 [确认高位拒绝价格]
  - RSI(14) > 75 且出现明显的顶背离
  - 成交量在流星形成当天巨量放大 [提示机构派发]
- **综合公式**：
  ```
  见顶信号 = 流星 AND (
             (随后日收盘价 < 流星实体低点) OR 
             (价格触及显著阻力位 且 ATR 高位回落)
           )
  ```

## 注意事项
- 需要确认流星的形态。
- 建议结合整体市场环境。
- 并非100%准确，应与其他分析工具结合。
