# CDLENGULFING（吞没模式）模式总结

## 模式描述
- 吞没模式是一个反转烛台模式，可以是看涨或看跌。
- 由两根蜡烛组成：
  - 看涨吞没：第二根长阳线完全吞没第一根阴线。
  - 看跌吞没：第二根长阴线完全吞没第一根阳线。
- 表明趋势可能反转。

## 使用时机
- 当市场处于趋势末端时，看到吞没模式可能预示反转。
- 适合在支撑或阻力位附近使用。
- 建议在日线图上使用。

## 配合指标
- **趋势确认**：与移动平均线配合。
- **动量指标**：RSI在超买/超卖区增强信号。
- **波动率指标**：ATR评估波动性。
- **成交量**：成交量放大时增强可靠性。
- **其他烛台模式**：可与其他反转模式结合。

## 量化识别信号
基于吞没模式及其他指标，其看涨（企稳）与看跌（见顶）的量化触发条件如下：

### 1. 看涨信号 (企稳)
- **基本条件**：看涨吞没形态出现（CDLENGULFING == 100）
- **综合公式**：
  ```
  企稳信号 = 看涨吞没 AND (
             (RSI < 30 且 第二根阳线成交量 > 第一根) OR 
             (价格向上突破 20日均线 且 MACD 金叉)
           )
  ```

### 2. 看跌信号 (见顶)
- **基本条件**：看跌吞没形态出现（CDLENGULFING == -100）
- **综合公式**：
  ```
  见顶信号 = 看跌吞没 AND (
             (RSI > 70 且 第二根阴线成交量显著放大) OR 
             (价格触及布林线上轨 且 ATR 高位回落)
           )
  ```

## 注意事项
- 需要确认吞没的完整性。
- 建议结合整体市场环境。
- 并非100%准确，应与其他分析工具结合。
