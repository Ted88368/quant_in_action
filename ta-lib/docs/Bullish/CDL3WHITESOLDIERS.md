# CDL3WHITESOLDIERS（三白兵）模式总结

## 模式描述
- 三白兵是一个看涨的反转烛台模式，通常出现在下跌趋势的底部。
- 由三根蜡烛组成：
  1. 第一根：阳线。
  2. 第二根：阳线，开盘价高于第一根的收盘价，收盘价高于第一根。
  3. 第三根：阳线，开盘价高于第二根的收盘价，收盘价高于第二根。
- 显示出市场从悲观到乐观的强劲转变。

## 使用时机
- 当市场处于下跌趋势时，看到三白兵模式，可能预示着价格反转上涨。
- 适合在中期趋势中识别可能的底部。
- 建议在低位区域使用。

## 配合指标
- **趋势确认**：与移动平均线（如SMA、EMA）配合，确认整体下跌趋势。
- **动量指标**：RSI（相对强弱指数）在30以下时增强看涨信号；MACD在低位金叉时配合。
- **波动率指标**：ATR（平均真实波幅）评估波动性。
- **成交量**：OBV（平衡成交量）上升时增强可靠性。
- **其他烛台模式**：可与Morning Star等其他看涨模式结合使用。

## 判断下跌企稳的条件
基于三白兵模式及其他指标，判断下跌是否企稳的综合条件可总结为以下公式：

- **基本条件**：三白兵形态出现（CDL3WHITESOLDIERS == 100）
- **增强条件**：
  - 前期处于明显的超卖区域（RSI < 30）
  - 三根阳线的成交量呈递增趋势
  - 价格突破重要压力位（如 20日均线）
- **综合公式**：
  ```
  企稳信号 = (三白兵 AND 成交量递增) OR 
             (三白兵 AND 价格突破 20日均线 AND RSI > 40)
  ```

## 注意事项
- 这是一个反转信号，但需要结合整体市场环境。
- 建议在日线或周线图上使用。
- 并非100%准确，应与其他分析工具结合。
