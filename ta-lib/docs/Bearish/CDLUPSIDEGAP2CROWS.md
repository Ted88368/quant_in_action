# CDLUPSIDEGAP2CROWS（上跳空乌鸦）模式总结

## 模式描述
- 上跳空乌鸦顶部反转，两根阴线跳空。

## 使用时机
- 上涨末端，卖出。

## 配合指标
- 成交量。
- MACD。

## 判断见顶回落的条件
基于上跳空两只乌鸦模式及其他指标，判断上涨趋势是否见顶的综合条件可总结为以下公式：

- **基本条件**：上跳空两只乌鸦形态出现（CDLUPSIDEGAP2CROWS == 100）
- **增强条件**：
  - 第三根阴线填补了第一根和第二根之间的跳空缺口 [确认反弹失败]
  - RSI(14) > 70 且处于上涨趋势后期
  - 成交量在阴线形成时伴随放大
- **综合公式**：
  ```
  见顶信号 = 上跳空两只乌鸦 AND (
             (价格跌破第一根阳线的最高点) OR 
             (MACD 出现死叉 且 价格处于密集成金区顶部)
           )
  ```

## 注意事项
