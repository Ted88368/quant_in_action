# CDLLADDERBOTTOM（梯底）模式总结

## 模式描述
- 梯底是一个多K线组成的看涨反转模式，通常出现在长期下跌趋势中。
- 由五根蜡烛组成：
  1. 前三根：持续下跌的阴线，且每一根的开盘价和收盘价都低于前一根。
  2. 第四根：带有较长上影线的阴线或十字星。
  3. 第五根：大阳线，开盘价高于前一根的实体，收盘价显著走高。
- 它显示了市场在极度悲观后，空头动能衰竭并由多头掌控局面。

## 使用时机
- 下跌趋势末端，作为强烈的底部反转信号。
- 适合在市场出现恐慌性抛售后的修复阶段寻找介入点。

## 配合指标
- **成交量**：第五根大阳线必须伴随成交量的显著放大以确认反转。
- **动量指标**：RSI 在 20 附近的严重超卖区开始转强。
- **支撑位**：与前期历史大底或重要技术支撑位重合。

## 判断下跌企稳的条件
基于梯底模式及其他指标，判断下跌是否企稳的综合条件可总结为以下公式：

- **基本条件**：梯底形态出现（CDLLADDERBOTTOM == 100）
- **增强条件**：
  - 最后三根阴线的实体逐渐缩小，且下影线表现出支撑性
  - 最后一根大阳线配合成交量急剧放大（放量突破）
  - RSI(14) 从 20 以下低位快速反弹并突破 30
- **综合公式**：
  ```
  企稳信号 = (梯底形态 AND 最后一根阳线放量) OR 
             (梯底形态 AND 价格站上 10日均线 AND RSI > 30)
  ```

## 注意事项
- 模式复杂，需五根K线完全成型。
- 阳线的反击力度（长度和量能）是决定成败的核心。
- 建议结合周线级别的大趋势分析。
