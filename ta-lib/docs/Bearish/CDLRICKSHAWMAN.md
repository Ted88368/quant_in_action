# CDLRICKSHAWMAN（长脚十字）模式总结

## 模式描述
- 长脚十字是一种特殊的十字星，上影线和下影线都非常长，表示多空双方在极广的价格范围内激烈争夺，最终回到原点，预示市场极度不确定，常为转势信号。

## 使用时机
- 出现在上涨趋势高位，预示转折。

## 配合指标
- RSI：处于 70 以上超买区。
- 成交量：配合高额成交量表示博弈激烈。

## 判断见顶回落的条件
基于长脚十字模式及其他指标，判断上涨趋势是否见顶的综合条件可总结为以下公式：

- **基本条件**：长脚十字形态出现（CDLRICKSHAWMAN == 100）
- **增强条件**：
  - RSI(14) 从超买区域向下掉头
  - 随后一个交易日收盘价跌破长脚十字的最低价 [确认为空头占优]
  - 价格处于长期均线或其他重要阻力位下方
- **综合公式**：
  ```
  见顶信号 = 长脚十字 AND (
             (随后日价格跌破十字星低点) OR 
             (成交量放大 且 价格触及布林线上轨)
           )
  ```
