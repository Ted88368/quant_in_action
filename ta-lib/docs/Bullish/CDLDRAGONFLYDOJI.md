# CDLDRAGONFLYDOJI（龙飞十字星）模式总结

## 模式描述
- 龙飞十字星是一种底部反转烛台模式，通常出现在下跌趋势的末端。该模式由一根十字星组成，其中开盘价和收盘价相等，位于价格区间的下端，下影线较长，表示买方力量逐渐增强。

## 使用时机
- 当市场处于下跌趋势中，出现龙飞十字星时，可视为潜在的买入信号，尤其是在重要支撑位附近。

## 配合指标
- 成交量：确认反转时成交量放大。
- RSI：低于30时增强信号。

## 判断下跌企稳的条件
基于龙飞十字星模式及其他指标，判断下跌是否企稳的综合条件可总结为以下公式：

- **基本条件**：龙飞十字星形态出现（CDLDRAGONFLYDOJI == 100）
- **增强条件**：
  - RSI(14) < 30 或处于近期低点
  - 价格触及布林线下轨
  - 下影线长度至少是实体的3倍（在十字星中实体极小）
- **综合公式**：
  ```
  企稳信号 = (龙飞十字星 AND RSI < 30) OR 
             (龙飞十字星 AND 价格触及支撑位 AND 成交量放大)
  ```

## 注意事项
- 需要后续K线确认反转，避免孤立使用；市场环境和整体趋势很重要。
