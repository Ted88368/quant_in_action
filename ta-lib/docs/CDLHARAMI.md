# CDLHARAMI（孕线模式）模式总结

## 模式描述
- 孕线模式是一个反转烛台模式，可以是看涨或看跌。
- 由两根蜡烛组成：
  - 看涨孕线：第一根长蜡烛，第二根小蜡烛完全包含在第一根内。
  - 看跌孕线：类似，但方向相反。
- 表明趋势可能反转。

## 使用时机
- 当市场处于趋势末端时，看到孕线模式可能预示反转。
- 适合在支撑或阻力位附近使用。
- 建议在日线图上使用。

## 配合指标
- **趋势确认**：与移动平均线配合。
- **动量指标**：RSI在超买/超卖区增强信号。
- **波动率指标**：ATR评估波动性。
- **成交量**：成交量放大时增强可靠性。
- **其他烛台模式**：可与其他反转模式结合。

## 注意事项
- 需要确认孕线的完整性。
- 建议结合整体市场环境。
- 并非100%准确，应与其他分析工具结合。
