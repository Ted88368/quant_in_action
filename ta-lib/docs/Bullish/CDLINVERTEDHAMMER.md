# CDLINVERTEDHAMMER（倒锤头）模式总结

## 模式描述
- 倒锤头是一种底部反转信号，上影线长，实体小，表示买方尝试。

## 使用时机
- 下跌趋势末端，买入信号。

## 配合指标
- RSI：底部反弹。
- 成交量：放大。

## 判断下跌企稳的条件
基于倒锤头模式及其他指标，判断下跌是否企稳的综合条件可总结为以下公式（逻辑组合，非严格数学公式，仅供参考）：

- **基本条件**：倒锤头形态出现（小实体 + 上影线 ≥ 实体2倍 + 下影线短/无）
- **增强条件**：
  - RSI < 30（超卖区）
  - MACD金叉（DIF上穿DEA）
  - 成交量放大（相对前几日）
  - 价格触及支撑位（前期低点或均线）
- **综合公式**：
  ```
  企稳信号 = 倒锤头形态 AND (
             (随后日收盘为阳线 且 收盘价 > 倒锤头最高价 [强确认]) OR 
             (RSI < 25 且 价格处于 250日等长期均线支撑位)
           )
  ```
  - 若满足任意组合，概率较高；结合多个指标更可靠。
  - 建议在量化策略中用Python验证（如talib计算指标）。

## 注意事项
- 需确认；上影线长度重要。
