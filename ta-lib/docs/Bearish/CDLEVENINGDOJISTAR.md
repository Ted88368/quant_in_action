# CDLEVENINGDOJISTAR（黄昏十字星）模式总结

## 模式描述
- 黄昏十字星是一种顶部反转模式，由三根K线组成：第一根阳线，第二根十字星，第三根阴线。十字星的开盘价和收盘价相等，位于高位，表示卖方力量开始占据优势。

## 使用时机
- 出现在上涨趋势末端，作为卖出信号。

## 配合指标
- MACD：顶背离时增强。
- 布林带：价格触及上轨时。

## 判断见顶回落的条件
基于黄昏十字星模式及其他指标，判断上涨趋势是否见顶的综合条件可总结为以下公式：

- **基本条件**：黄昏十字星形态出现（CDLEVENINGDOJISTAR == 100）
- **增强条件**：
  - 第三根阴线的成交量大于第一根阳线 [确认由于多方动能衰竭后的空方反扑]
  - RSI(14) 从 70 以上高位回落
  - 十字星位置处于长期压力线附近
- **综合公式**：
  ```
  见顶信号 = 黄昏十字星 AND (
             (随后日价格跌破十字星最低点) OR 
             (MACD DIF 线下穿 DEA 线 且 价格处于阻力位)
           )
  ```

## 注意事项
- 需观察后续走势确认，避免在震荡市中使用；成交量萎缩时信号更强。
