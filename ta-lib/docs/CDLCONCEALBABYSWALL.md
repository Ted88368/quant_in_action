# CDLCONCEALBABYSWALL（藏婴吞没）模式总结

## 模式描述
- 藏婴吞没是一个罕见的看跌反转烛台模式，通常出现在上涨趋势中。
- 由四根蜡烛组成：
  1. 第一根：阳线。
  2. 第二根：阴线，有长上影线。
  3. 第三根：阴线，有长上影线，收盘价低于第二根。
  4. 第四根：长阴线，吞没前三根。
- 显示出市场从上涨到下跌的转变。

## 使用时机
- 当市场处于上涨趋势时，看到藏婴吞没模式，可能预示着价格反转下跌。
- 适合在中期趋势中识别可能的顶部。
- 建议在高位区域使用。

## 配合指标
- **趋势确认**：与移动平均线（如SMA、EMA）配合，确认整体上涨趋势。
- **动量指标**：RSI（相对强弱指数）在70以上时增强看跌信号；MACD在高位死叉时配合。
- **波动率指标**：ATR（平均真实波幅）评估波动性。
- **成交量**：OBV（平衡成交量）下降时增强可靠性。
- **其他烛台模式**：可与Three Black Crows等其他看跌模式结合使用。

## 注意事项
- 这是一个罕见模式，反转信号较强，但需要结合整体市场环境。
- 建议在日线或周线图上使用。
- 并非100%准确，应与其他分析工具结合。
