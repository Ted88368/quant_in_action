# CDL3INSIDE（三内部）模式总结

## 模式描述
- 三内部是一个反转烛台模式，可以是看涨或看跌，取决于第三根蜡烛的方向。
- 由三根蜡烛组成：
  1. 第一根：长蜡烛（阳或阴）。
  2. 第二根：小蜡烛，完全包含在第一根的范围内。
  3. 第三根：长蜡烛，与第一根方向相反，收盘价超出第一根的范围。
- 如果第三根是阳线，则为看涨三内部；如果是阴线，则为看跌三内部。

## 使用时机
- 当市场处于趋势中时，看到三内部模式，可能预示着价格反转。
- 适合在中期趋势中识别可能的转折点。
- 建议在支撑或阻力区域使用。

## 配合指标
- **趋势确认**：与移动平均线（如SMA、EMA）配合，确认整体趋势。
- **动量指标**：RSI（相对强弱指数）在超买/超卖区时增强信号；MACD交叉时配合。
- **波动率指标**：ATR（平均真实波幅）评估波动性。
- **成交量**：OBV（平衡成交量）变化时增强可靠性。
- **其他烛台模式**：可与其他反转模式结合使用。

## 量化识别信号
基于三内部模式及其他指标，其看涨（企稳）与看跌（见顶）的量化触发条件如下：

### 1. 看涨信号 (企稳)
- **基本条件**：看涨三内部形态出现（CDL3INSIDE == 100）
- **综合公式**：
  ```
  企稳信号 = 看涨三内部 AND (
             (RSI < 30 且 价格触及布林线下轨) OR 
             (收盘价突破第一根长阴线的高点 且 成交量放大)
           )
  ```

### 2. 看跌信号 (见顶)
- **基本条件**：看跌三内部形态出现（CDL3INSIDE == -100）
- **综合公式**：
  ```
  见顶信号 = 看跌三内部 AND (
             (RSI > 70 且 价格处于前期高点阻力位) OR 
             (收盘价跌破第一根长阳线的低点 且 MACD 死叉)
           )
  ```

## 注意事项
- 这是一个反转信号，但需要确认第三根蜡烛的方向。
- 建议在日线或周线图上使用。
- 并非100%准确，应与其他分析工具结合。
