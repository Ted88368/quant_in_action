# CDLDOJI（十字星）模式总结

## 模式描述
- 十字星是一个中性烛台模式，表示市场犹豫不决。
- 蜡烛的开盘价和收盘价非常接近，几乎相等，上下影线较长。
- 表明买卖双方力量均衡，可能预示趋势反转或延续。

## 使用时机
- 当市场处于趋势中时，看到十字星可能预示反转。
- 适合在关键支撑或阻力位附近使用。
- 建议结合趋势和成交量分析。

## 配合指标
- **趋势确认**：与移动平均线配合，确认趋势方向。
- **动量指标**：RSI在50附近时增强信号。
- **波动率指标**：ATR评估波动性。
- **成交量**：成交量放大时增强可靠性。
- **其他烛台模式**：可与Hammer（锤头）等结合。

## 量化识别信号
基于十字星模式及其他指标，其作为变盘信号的量化触发条件如下：

### 1. 看涨确认 (企稳)
- **基本条件**：十字星形态出现（CDLDOJI == 100）
- **综合公式**：
  ```
  企稳信号 = 十字星 AND (
             (随后日收盘价突破十字星最高价 且 成交量放大) OR 
             (RSI < 30 且 价格处于显著波段低点)
           )
  ```

### 2. 看跌确认 (见顶)
- **基本条件**：十字星形态出现（CDLDOJI == 100）
- **综合公式**：
  ```
  见顶信号 = 十字星 AND (
             (随后日收盘价跌破十字星最低价 且 MACD 死叉) OR 
             (RSI > 70 且 价格处于布林线上轨)
           )
  ```

## 注意事项
- 十字星本身不提供方向信号，需要上下文。
- 建议在较高时间框架使用。
- 并非100%准确，应与其他分析工具结合。
