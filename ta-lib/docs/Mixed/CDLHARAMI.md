# CDLHARAMI（孕线模式）模式总结

## 模式描述
- 孕线模式是一个反转烛台模式，可以是看涨或看跌。
- 由两根蜡烛组成：
  - 看涨孕线：第一根长蜡烛，第二根小蜡烛完全包含在第一根内。
  - 看跌孕线：类似，但方向相反。
- 表明趋势可能反转。

## 使用时机
- 当市场处于趋势末端时，看到孕线模式可能预示反转。
- 适合在支撑或阻力位附近使用。
- 建议在日线图上使用。

## 配合指标
- **趋势确认**：与移动平均线配合。
- **动量指标**：RSI在超买/超卖区增强信号。
- **波动率指标**：ATR评估波动性。
- **成交量**：成交量放大时增强可靠性。
- **其他烛台模式**：可与其他反转模式结合。

## 量化识别信号
基于孕线模式及其他指标，其看涨（企稳）与看跌（见顶）的量化触发条件如下：

### 1. 看涨信号 (企稳)
- **基本条件**：看涨孕线形态出现（CDLHARAMI == 100）
- **综合公式**：
  ```
  企稳信号 = 看涨孕线 AND (
             (随后一个交易日收阳 且 突破第一根大阴线的中点) OR 
             (RSI 出现底背离 且 价格处于长期支撑位)
           )
  ```

### 2. 看跌信号 (见顶)
- **基本条件**：看跌孕线形态出现（CDLHARAMI == -100）
- **综合公式**：
  ```
  见顶信号 = 看跌孕线 AND (
             (随后一个交易日收阴 且 跌破该形态的最低价) OR 
             (RSI > 70 且 价格处于高位震荡阻力区)
           )
  ```

## 注意事项
- 需要确认孕线的完整性。
- 建议结合整体市场环境。
- 并非100%准确，应与其他分析工具结合。
