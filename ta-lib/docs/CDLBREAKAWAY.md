# CDLBREAKAWAY（脱离）模式总结

## 模式描述
- 脱离是一个反转烛台模式，可以是看涨或看跌，取决于第五根蜡烛的方向。
- 由五根蜡烛组成：
  1. 第一根：长蜡烛，与趋势方向一致。
  2. 第二根：蜡烛，跳空同向。
  3. 第三根：小蜡烛。
  4. 第四根：蜡烛，跳空远离第三根。
  5. 第五根：长蜡烛，与第一根方向相反。
- 显示出市场从趋势中脱离。

## 使用时机
- 当市场处于趋势中时，看到脱离模式，可能预示着价格反转。
- 适合在中期趋势中识别可能的转折点。
- 建议在支撑或阻力区域使用。

## 配合指标
- **趋势确认**：与移动平均线（如SMA、EMA）配合，确认整体趋势。
- **动量指标**：RSI（相对强弱指数）在超买/超卖区时增强信号；MACD交叉时配合。
- **波动率指标**：ATR（平均真实波幅）评估波动性。
- **成交量**：OBV（平衡成交量）变化时增强可靠性。
- **其他烛台模式**：可与其他反转模式结合使用。

## 注意事项
- 这是一个复杂模式，反转信号较强，但需要确认第五根蜡烛的方向。
- 建议在日线或周线图上使用。
- 并非100%准确，应与其他分析工具结合。
