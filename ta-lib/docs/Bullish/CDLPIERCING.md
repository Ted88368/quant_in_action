# CDLPIERCING（刺穿模式）模式总结

## 模式描述
- 刺穿模式是一个看涨的反转烛台模式，通常出现在下跌趋势中。
- 由两根蜡烛组成：
  1. 第一根：长阴线，表示强劲下跌。
  2. 第二根：跳空低开的阳线，收盘价深入第一根阴线的实体，至少超过50%。

## 使用时机
- 当市场处于下跌趋势时，看到刺穿模式可能预示反弹。
- 适合在中期趋势中识别可能的底部。
- 建议在低位区域或超卖区使用。

## 配合指标
- **趋势确认**：与移动平均线配合，确认下跌趋势。
- **动量指标**：RSI在30以下时增强信号；MACD在低位金叉时配合。
- **波动率指标**：ATR评估波动性。
- **成交量**：OBV上升时增强可靠性。
- **其他烛台模式**：可与Hammer（锤头）等其他看涨模式结合。

## 判断下跌企稳的条件
基于刺穿模式及其他指标，判断下跌是否企稳的综合条件可总结为以下公式（逻辑组合，非严格数学公式，仅供参考）：

- **基本条件**：刺穿形态出现（长阴线 + 跳空低开阳线，收盘深入阴线实体 ≥ 50%）
- **增强条件**：
  - RSI < 30（超卖区）
  - MACD金叉（DIF上穿DEA）
  - 成交量放大（相对前几日）
  - 价格触及支撑位（前期低点或均线）
- **综合公式**：
  ```
  企稳信号 = 刺穿形态 AND (
             (价格触及显著的历史支撑位 且 RSI < 30) OR 
             (成交量显著放大 且 MACD 柱线由负转正)
           )
  ```
  - 若满足任意组合，概率较高；结合多个指标更可靠。
  - 建议在量化策略中用Python验证（如talib计算指标）。

## 注意事项
- 这是一个反转信号，但需要结合整体市场环境。
- 建议在日线或周线图上使用。
- 并非100%准确，应与其他分析工具结合。
