# CDLMATCHINGLOW（匹配低点）模式总结

## 模式描述
- 匹配低点是一个两K线组成的看涨反转模式，通常出现在下跌趋势中。
- 由两根阴线组成：
  1. 第一根：大阴线。
  2. 第二根：阴线，其收盘价与第一根阴线的收盘价几乎相同。
- 虽然两根都是阴线，但收盘价未能进一步下探，显示出在该水平位有强力支撑，可能预示着底部的形成。

## 使用时机
- 下跌趋势的末端，作为寻找支撑位和反转可能的信号。
- 适合在价格测试前期低点或重要心理价位（整数位）时使用。

## 配合指标
- **支撑线**：该模式的收盘价水平应与历史支撑位或长期趋势线重合。
- **动量指标**：RSI(14) 处于 30 以下或出现底背离（价格创新低而 RSI 未创新低）。
- **成交量**：第二根阴线的成交量通常应小于第一根，显示抛盘力竭。

## 判断下跌企稳的条件
基于匹配低点模式及其他指标，判断下跌是否企稳的综合条件可总结为以下公式：

- **基本条件**：匹配低点形态出现（CDLMATCHINGLOW == 100）
- **增强条件**：
  - 两个低点恰好位于长期支撑位或关键整数关口
  - 第二根阴线的成交量显著小于第一根（杀跌动能衰竭）
  - MACD 出现底背离迹象，DIF 趋于走平
- **综合公式**：
  ```
  企稳信号 = 匹配低点 AND (
             (随后日收盘价突破前二日阴线的最高价 [强确认]) OR 
             (第二根阴线成交量极度萎缩 且 RSI 出现底背离)
           )
  ```

## 注意事项
- 该模式通常被视为低可靠性，必须结合后续K线的确认。
- 确认信号通常是第三根蜡烛以长阳线收盘，且收盘价高于前两根阴线的实体。
- 价格不应在短时间内跌破匹配的低点。
