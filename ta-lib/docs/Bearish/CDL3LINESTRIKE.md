# CDL3LINESTRIKE（三线打击）模式总结

## 模式描述
- 三线打击是一个看跌的反转烛台模式，通常出现在上涨趋势中。
- 由四根蜡烛组成：
  1. 第一根：阳线。
  2. 第二根：阳线，开盘价高于第一根的收盘价。
  3. 第三根：阳线，开盘价高于第二根的收盘价。
  4. 第四根：长阴线，开盘价高于第三根的收盘价，收盘价低于前三根的最低点。
- 显示出市场从上涨到突然下跌的转变。

## 使用时机
- 当市场处于上涨趋势时，看到三线打击模式，可能预示着价格反转下跌。
- 适合在中期趋势中识别可能的顶部。
- 建议在高位区域使用。

## 配合指标
- **趋势确认**：与移动平均线（如SMA、EMA）配合，确认整体上涨趋势。
- **动量指标**：RSI（相对强弱指数）在70以上时增强看跌信号；MACD在高位死叉时配合。
- **波动率指标**：ATR（平均真实波幅）评估波动性。
- **成交量**：OBV（平衡成交量）下降时增强可靠性。
- **其他烛台模式**：可与Three Black Crows等其他看跌模式结合使用。

## 判断见顶回落的条件
基于三线打击模式及其他指标，判断上涨趋势是否见顶的综合条件可总结为以下公式：

- **基本条件**：三线打击形态出现（CDL3LINESTRIKE == 100）
- **增强条件**：
  - 第四根大阴线的成交量是前三根均值的 2 倍以上
  - 价格一举跌破过去 5 个交易日的最低点
  - RSI 从 70 以上高位快速滑落
- **综合公式**：
  ```
  见顶信号 = 三线打击 AND (
             (成交量激增 [确认机构出货]) OR 
             (价格下穿 10日均线 且 MACD 柱线走弱)
           )
  ```

## 注意事项
- 这是一个反转信号，但需要结合整体市场环境。
- 建议在日线或周线图上使用。
- 并非100%准确，应与其他分析工具结合。
