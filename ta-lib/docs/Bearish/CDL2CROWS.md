# CDL2CROWS（两只乌鸦）模式总结

## 模式描述
- 两只乌鸦是一个看跌的反转烛台模式，通常出现在上涨趋势中。
- 由三根蜡烛组成：
  1. 第一根：长阳线，表示强劲上涨。
  2. 第二根：跳空高开的阴线，收盘价仍高于开盘价。
  3. 第三根：阴线，收盘价低于第二根的收盘价。

## 使用时机
- 当市场处于上涨趋势时，看到两只乌鸦模式，可能预示着价格反转下跌。
- 适合在中期趋势中识别可能的顶部。
- 建议在高位区域或超买区使用，以增加可靠性。

## 配合指标
- **趋势确认**：与移动平均线（如SMA、EMA）配合，确认整体上涨趋势。
- **动量指标**：RSI（相对强弱指数）在70以上时增强看跌信号；MACD在高位死叉时配合。
- **波动率指标**：ATR（平均真实波幅）评估波动性，避免在低波动期误判。
- **成交量**：OBV（平衡成交量）下降时增强可靠性。
- **其他烛台模式**：可与Evening Star（黄昏之星）等其他看跌模式结合使用。

## 判断见顶回落的条件
基于两只乌鸦模式及其他指标，判断上涨趋势是否见顶的综合条件可总结为以下公式：

- **基本条件**：两只乌鸦形态出现（CDL2CROWS == 100）
- **增强条件**：
  - RSI(14) > 70（显著超买状态）
  - 第三根阴线的收盘价跌破第一根阳线的中点
  - MACD 出现死叉或 DIF 线动能开始衰减
- **综合公式**：
  ```
  见顶信号 = 两只乌鸦 AND (
             (RSI > 70 且 价格跌破 5日均线) OR 
             (成交量显著萎缩 且 随后日继续收阴)
           )
  ```

## 注意事项
- 这是一个反转信号，但需要结合整体市场环境和风险管理。
- 建议在日线或周线图上使用，避免在低时间框架上的噪音。
- 并非100%准确，应与其他分析工具结合。
