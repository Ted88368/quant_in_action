# CDLIDENTICAL3CROWS（三只乌鸦）模式总结

## 模式描述
- 三只乌鸦由三根连续阴线组成，每根开盘价在前一根收盘价附近，收盘价递减，表示熊市。

## 使用时机
- 上涨趋势末端，卖出信号。

## 配合指标
- 成交量：放大。
- MACD：死叉。

## 判断见顶回落的条件
基于三只乌鸦（同等开盘）模式及其他指标，判断上涨趋势是否见顶的综合条件可总结为以下公式：

- **基本条件**：三只乌鸦形态出现（CDLIDENTICAL3CROWS == 100）
- **增强条件**：
  - 三根阴线的开盘价几乎相等，且都在前一根的收盘价附近 [展示极强的空方统治力]
  - 三日累计跌幅巨大，且成交量呈递增趋势
  - 价格有效跌破 20日均线或上涨趋势线
- **综合公式**：
  ```
  见顶信号 = 三只乌鸦 AND (
             (成交量显著增加) OR 
             (RSI 下穿 50 且 价格处于高位破位状态)
           )
  ```

## 注意事项
- 需三根完整阴线；避免在底部。
