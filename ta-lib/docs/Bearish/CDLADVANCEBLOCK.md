# CDLADVANCEBLOCK（推进块）模式总结

## 模式描述
- 推进块是一个看跌的延续或反转烛台模式，通常出现在上涨趋势中。
- 由三根蜡烛组成：
  1. 第一根：强阳线。
  2. 第二根：阳线，开盘价高于第一根的收盘价，但上影线较长。
  3. 第三根：阳线，开盘价高于第二根的收盘价，上影线更长，显示出上涨动能减弱。
- 显示出市场上涨动力逐渐减弱。

## 使用时机
- 当市场处于上涨趋势时，看到推进块模式，可能预示着价格上涨放缓或反转下跌。
- 适合在中期趋势中识别可能的顶部。
- 建议在高位区域使用。

## 配合指标
- **趋势确认**：与移动平均线（如SMA、EMA）配合，确认整体上涨趋势。
- **动量指标**：RSI（相对强弱指数）在70以上时增强看跌信号；MACD在高位死叉时配合。
- **波动率指标**：ATR（平均真实波幅）评估波动性。
- **成交量**：OBV（平衡成交量）下降时增强可靠性。
- **其他烛台模式**：可与Three Black Crows等其他看跌模式结合使用。

## 判断见顶回落的条件
基于推进块模式及其他指标，判断上涨趋势是否见顶的综合条件可总结为以下公式：

- **基本条件**：推进块形态出现（CDLADVANCEBLOCK == 100）
- **增强条件**：
  - 三根阳线的实体依次缩小，而上影线依次增长 [上涨动能衰竭]
  - RSI(14) > 75 且出现顶背离迹象
  - 成交量在第三根阳线处显著萎缩（买盘不济）
- **综合公式**：
  ```
  见顶信号 = 推进块 AND (
             (随后一个交易日收盘跌破第三根阳线的最低价) OR 
             (价格触及布林线上轨 且 ATR 高位回落)
           )
  ```

## 注意事项
- 这是一个警告信号，上涨可能放缓。
- 建议在日线或周线图上使用。
- 并非100%准确，应与其他分析工具结合。
