# CDLHAMMER（锤头）模式总结

## 模式描述
- 锤头是一个看涨的反转烛台模式，通常出现在下跌趋势中。
- 蜡烛的特点：小实体，下影线长（至少为实体的2倍），上影线短或无。
- 表明在低点有买方力量，价格可能反弹。

## 使用时机
- 当市场处于下跌趋势时，看到锤头可能预示反弹。
- 适合在支撑位附近使用。
- 建议在日线图上使用。

## 配合指标
- **趋势确认**：与移动平均线配合，确认下跌趋势。
- **动量指标**：RSI在30以下时增强信号。
- **波动率指标**：ATR评估波动性。
- **成交量**：成交量放大时增强可靠性。
- **其他烛台模式**：可与Morning Star（启明星）结合。

## 判断下跌企稳的条件
基于锤头模式及其他指标，判断下跌是否企稳的综合条件可总结为以下公式（逻辑组合，非严格数学公式，仅供参考）：

- **基本条件**：锤头形态出现（小实体 + 下影线 ≥ 实体2倍 + 上影线短/无）
- **增强条件**：
  - RSI < 30（超卖区）
  - MACD金叉（DIF上穿DEA）
  - 成交量放大（相对前几日）
  - 价格触及支撑位（前期低点或均线）
- **综合公式**：
  ```
  企稳信号 = 锤头形态 AND (
             (ATR 在高位回落 [确认波动减小]) OR 
             (成交量显著放大 AND 随后日收盘价突破锤头实体)
           )
  ```
  - 若满足任意组合，概率较高；结合多个指标更可靠。
  - 建议在量化策略中用Python验证（如talib计算指标）。

## 注意事项
- 需要确认锤头的形态。
- 建议结合整体市场环境。
- 并非100%准确，应与其他分析工具结合。
