# CDLCLOSINGMARUBOZU（收盘光头光脚）模式总结

## 模式描述
- 收盘光头光脚是一个延续烛台模式，可以是看涨或看跌，取决于蜡烛的颜色。
- 由一根蜡烛组成：
  - 看涨：阳线，无影线，收盘价为最高价。
  - 看跌：阴线，无影线，收盘价为最低价。
- 显示出市场强力延续趋势。

## 使用时机
- 当市场处于趋势中时，看到收盘光头光脚模式，可能预示着价格延续趋势。
- 适合在中期趋势中确认趋势强度。
- 建议在趋势中段使用。

## 配合指标
- **趋势确认**：与移动平均线（如SMA、EMA）配合，确认整体趋势。
- **动量指标**：RSI（相对强弱指数）在趋势方向时增强信号；MACD在趋势方向时配合。
- **波动率指标**：ATR（平均真实波幅）评估波动性。
- **成交量**：OBV（平衡成交量）在趋势方向时增强可靠性。
- **其他烛台模式**：可与其他延续模式结合使用。

## 注意事项
- 这是一个延续信号，不是反转。
- 建议在日线或周线图上使用。
- 并非100%准确，应与其他分析工具结合。
