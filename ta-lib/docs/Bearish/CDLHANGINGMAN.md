# CDLHANGINGMAN（上吊线）模式总结

## 模式描述
- 上吊线是一种顶部反转模式，实体小，下影线长，表示卖方力量在高位出现。

## 使用时机
- 出现在上涨趋势末端，作为卖出信号。

## 配合指标
- 支撑阻力：重要阻力位。
- 成交量：萎缩时。

## 判断见顶回落的条件
基于上吊线模式及其他指标，判断上涨趋势是否见顶的综合条件可总结为以下公式：

- **基本条件**：上吊线形态出现（CDLHANGINGMAN == 100）
- **增强条件**：
  - 随后一个交易日的收盘价远低于上吊线的实体 [确认为见顶确认烛]
  - RSI(14) > 70 且开始转向下行
  - 上吊线形成时成交量较前几日显著萎缩（提示多头后续乏力）
- **综合公式**：
  ```
  见顶信号 = 上吊线 AND (
             (随后日收盘价 < 上吊线实体低点) OR 
             (MACD 出现死叉 且 价格处于前期高点密集区)
           )
  ```

## 注意事项
- 需确认后续走势；实体颜色不重要。
