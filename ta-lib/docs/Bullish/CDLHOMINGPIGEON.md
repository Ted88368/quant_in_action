# CDLHOMINGPIGEON（归巢鸽）模式总结

## 模式描述
- 归巢鸽是一种看涨反转模式，通常出现在下跌趋势中。
- 由两根阴线组成：
  1. 第一根：大阴线。
  2. 第二根：小阴线，其实体完全被包含在第一根阴线的收盘价和开盘价之间。
- 这类似于看跌的“身怀六甲”（Harami），但在这种情况下，两根都是阴线。它显示出卖压正在减弱。

## 使用时机
- 当市场处于下跌趋势末端时，归巢鸽可能预示着趋势的衰减和潜在的反弹。
- 适合在多头寻找底部反转信号时使用。

## 配合指标
- **成交量**：第一根大阴线通常伴随较高成交量，第二根成交量显著萎缩则增强信号。
- **动量指标**：RSI 在 30 以下或 MACD 柱线缩短。
- **均线支撑**：如果出现在长期移动平均线（如 120日、250日线）附近。

## 判断下跌企稳的条件
基于归巢鸽模式及其他指标，判断下跌是否企稳的综合条件可总结为以下公式：

- **基本条件**：归巢鸽形态出现（CDLHOMINGPIGEON == 100）
- **增强条件**：
  - 第一根大阴线的成交量处于近期高位（恐慌抛售阶段）
  - 第二根小阴线成交量萎缩（卖力枯竭）
  - RSI(14) 出现底背离
- **综合公式**：
  ```
  企稳信号 = (归巢鸽 AND 成交量确认) OR 
             (归巢鸽 AND 价格后续突破 5日均线 AND RSI > 35)
  ```

## 注意事项
- 需要后续K线确认（如同向阳线收盘价高于归巢鸽实体）。
- 二者的实体颜色必须为阴线（在某些定义中）。
- 信号强度通常中等，建议结合其他分析工具。
