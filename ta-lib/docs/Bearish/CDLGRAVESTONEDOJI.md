# CDLGRAVESTONEDOJI（墓碑十字星）模式总结

## 模式描述
- 墓碑十字星是一种顶部反转信号，开盘价和收盘价相等，位于价格区间上端，上影线较长，表示卖方力量增强。

## 使用时机
- 下跌趋势开始时，作为卖出信号。

## 配合指标
- RSI：高于70时。
- 成交量：放大时确认。

## 判断见顶回落的条件
基于墓碑十字星模式及其他指标，判断上涨趋势是否见顶的综合条件可总结为以下公式：

- **基本条件**：墓碑十字星形态出现（CDLGRAVESTONEDOJI == 100）
- **增强条件**：
  - RSI(14) > 70（极端超买）
  - 成交量在十字星形成当天显著放大 [提示高位抛压沉重]
  - 随后一个交易日为阴线，且收盘价低于十字星最低价
- **综合公式**：
  ```
  见顶信号 = 墓碑十字星 AND (
             (RSI > 75 且 价格触发布林线上轨) OR 
             (成交量放大 且 随后日跳空下行)
           )
  ```

## 注意事项
- 需要后续阴线确认；避免在低位使用。
