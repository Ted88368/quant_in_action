# CDLMORNINGDOJISTAR（启明星十字）模式总结

## 模式描述
- 启明星十字底部反转，三根K线：阴线、十字星、阳线。

## 使用时机
- 下跌末端，买入。

## 配合指标
- 成交量：放大。
- RSI。

## 判断下跌企稳的条件
基于启明星十字模式及其他指标，判断下跌是否企稳的综合条件可总结为以下公式（逻辑组合，非严格数学公式，仅供参考）：

- **基本条件**：启明星十字形态出现（阴线 + 十字星 + 阳线）
- **增强条件**：
  - RSI < 30（超卖区）
  - MACD金叉（DIF上穿DEA）
  - 成交量放大（相对前几日）
  - 价格触及支撑位（前期低点或均线）
- **综合公式**：
  ```
  企稳信号 = 启明星十字形态 AND (
             (第三根阳线收盘价突破第一根阴线的中点 [强确认]) OR 
             (第三根阳线放量突破 20日均线 且 MACD 金叉)
           )
  ```
  - 若满足任意组合，概率较高；结合多个指标更可靠。
  - 建议在量化策略中用Python验证（如talib计算指标）。

## 注意事项
- 需完整三根；确认信号。
